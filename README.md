# KH_MiniHittegods

## Start

Startes med: ´docker compose up --build´ fra MiniHittegodsApi katalogen.

## Scalar

Programmet bruker OpenApi med Scalar som interface, denne er tilgjengelig
på (http://localhost:8080/scalar)

## Tester

I katalogen Tester er det to prosjekter, den ene er for å teste
kjernefunksjonaliteten og den andre for webapien.

Disse kan kjøres med ´dotnet test´ eller helst med vscode sin test funksjonalitet.
